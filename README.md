# algoritmos-leetcode

Projeto: coleção de soluções para problemas do LeetCode

Este repositório é uma pasta de estudos para resolver desafios de algoritmo e estrutura de dados.
Cada pasta (`easy`, `medium`, `hard`) contém subprojetos e exemplos de código com base nas questões do LeetCode.

## Objetivo

- Praticar lógica e pensamento computacional.
- Documentar solução, análise de complexidade e testes locais.

## Estrutura do repositório

- `easy/` : problemas fáceis
- `medium/` : problemas intermediários
- `hard/` : problemas difíceis

Exemplo de subpasta com projeto:
- `medium/add-two-numbers/` (problema "Add Two Numbers")

## Como compilar e rodar

### Java

1. Abra terminal na pasta do problema (`algoritmos-leetcode/medium/add-two-numbers/src` etc.).
2. Compile:

   ```bash
   javac Main.java
   ```

3. Execute:

   ```bash
   java Main
   ```

## Observação

Este README descreve como usar o projeto de forma simplificada; adapte os comandos para cada subprojeto conforme linguagem e estrutura.
