# Add Two Numbers

## Problema
Somar dois números representados por linked lists...

## Ideia
Percorrer ambas listas simultaneamente somando dígitos e controlando o carry ("vai um").

## Complexidade
- Tempo: O(n)
- Espaço: O(n)

## Observações
- Uso de dummy node (nó fictício) para facilitar construção da lista
